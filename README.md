# -web-animation8

Faithful 1:1 clone of **https://www.vectrfl.com/** — Vectr, an Astro static site with a Three.js/WebGL 3D scene (Draco-compressed GLB models) + scroll animations.

- Runs locally on port 3700 (`npm run dev`).
- 7 pages; all same-origin assets self-hosted (CSS, images, GLB models, Draco decoder).
